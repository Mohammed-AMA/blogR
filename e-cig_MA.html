Proportions (%) Meta analysis

This is the proportions meta-analysis based on the number of events to the total of the sample. The #marked varible needs to be updated based on the data frame.

A Generalized linear mixed model (GLMM) was used to calculated the pooled proportions based on the recommendation of (https://pubmed.ncbi.nlm.nih.gov/30945438/).

  • first we need to call the dataframe from the dmetar library. The data is called OpioidMisuse
m1 <- OpioidMisuse
  • Now, lets have a look at the dataframe using the glimpse function from tidyverse package.
glimpse(m1) #      
## Rows: 15
## Columns: 3
## $ author <chr> "Becker, 2008", "Boyd, 2009", "Boyd, 2007", "Cerda, 2014", "Fie
## $ event  <dbl> 2186, 91, 126, 543, 6496, 10850, 86, 668, 843, 647, 11521, 1111
## $ n      <dbl> 21826, 912, 1084, 7646, 55215, 114783, 527, 9403, 11274, 8888,

  • we can see that this data m1 has 3 column and 15 rows. The columns represents the variables and the rows represents the observations.

  • Now lets pool (add) them together using the following function.

meta_prop <- metaprop(event = event, #number_of_events_var
                   n =  n, #total_n
                   studlab = author, #labeling variable
                   data = m1, ##data,
                   method = "GLMM",
                   sm = "PLOGIT",
                   fixed = FALSE,
                   random = TRUE,
                   hakn = TRUE,
                   title = "Prevleance of Misuse") #title of the model
  • Now we have run the function and we would like to see the results of our newly created object meta_prop

Forest plot - Proportions

To help us visualise the previous model, we need to make a forest plot. A basic forest, yet informative plot that can be made using the forest.meta function.

forest.meta(meta_prop)

Odds ratio (OR) Meta analysis

In this section an OR meta-analysis based on the number of events to the total of the sample in both study arms (ecig and controls). The #marked vaible needs to be updated based on the data.

This is a random effects model and utilised the Mantel-Haenszel method without continuity correction.

meta_or <- metabin(event.e = #number of events in e-cig, 
                 n.e = #total e-cig,
                 event.c = #number of events in control,
                 n.c = #total control,
                 studlab = author,
                 data = #data,
                 sm = "OR",
                 method = "MH",
                 MH.exact = TRUE,
                 fixed = FALSE,
                 random = TRUE,
                 method.tau = "PM",
                 hakn = TRUE,
                 title = "Odds ratio of #Symptom")
summary(meta_or)
